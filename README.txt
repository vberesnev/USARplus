Тестовое задание для ЮСАР+

Требуется разработать приложение, которое имеет одно окно с возможностью ввода сообщения и кнопку. По нажатию на кнопку происходит отправка и запоминание сообщения, момента сохранения сообщения, имени пользователя, который сохранил данные в БД, например, файл, либо LiteDB.

При повторном запуске текстовая область должна содержать предыдущее значение + данные о том, кто и когда его сохранил.

Если сообщение не заполнено, кнопка должна быть неактивна.

Реализовать задание необходимо с использованием C# (WPF/XAML/MVVM).

Если предложите варианты тестирования и документирования, будет плюсом.
